# 🎓 University Event Management System

A web-based platform that helps universities manage events efficiently by handling event creation, registration, ticketing, payments, notifications, feedback, and analytics.

## ✨ Features

* Event creation and management
* Online registration and ticketing
* Secure payment processing
* Automated notifications and reminders
* Lost & Found management
* Feedback and messaging system
* Attendance and revenue analytics
* Role-based access (Admin, Organizer, Attendee)

## 🛠 Technology Stack

* **Frontend:** React + Vite
* **Backend:** Node.js + Express.js
* **Database:** MongoDB
* **Authentication:** JWT

## 🚀 How It Works

1. Organizers create and publish events.
2. Attendees browse, register, and purchase tickets.
3. The system sends notifications and reminders.
4. Admins monitor attendance, revenue, and feedback through analytics.

## 🎯 Benefits

* Simplifies campus event management
* Improves attendee engagement
* Automates communication and ticketing
* Provides valuable event insights

## 📢 Elevator Pitch

A complete university event management platform that streamlines event creation, ticketing, communication, and event operations for students, organizers, and administrators.
